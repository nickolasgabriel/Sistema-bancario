# # 💰 Sistema Bancário em Java

Este é um projeto de um **Sistema Bancário** feito em Java com o objetivo de praticar os conceitos básicos da linguagem. Todo o sistema funciona pelo terminal e foi desenvolvido durante meus estudos.

## Funcionalidades

O sistema permite:

* Fazer login com usuário e senha;
* Consultar saldo;
* Fazer depósitos;
* Realizar saques;
* Transferir dinheiro por uma chave Pix;
* Ver o extrato das movimentações da conta.

## O que foi utilizado

Neste projeto utilizei apenas conceitos básicos de Java, como:

* Variáveis;
* `if`, `else` e `switch`;
* `for`, `while` e `do while`;
* Métodos;
* Classes;
* ArraysList;
* Scanner para entrada de dados;
* `LocalDateTime` para registrar a data e a hora das operações.

## Como executar

1. Clone este repositório.
2. Abra o projeto em uma IDE Java.
3. Execute a classe `SistemaBancario`.
